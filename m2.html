Сумма больше остатка долга

1. Изменить сумму возврата

Вернуться в форму. FamilyPilot подставит точный остаток долга.

2. Проценты / подарок

Основной долг закроется полностью. Превышение сохранится отдельной строкой «Проценты / подарок» без дополнительных вопросов.

3. Создать встречный долг

Основной долг закроется. На сумму превышения возникнет новый долг в обратную сторону с автоматическим комментарием о происхождении и дате.

4. Создать отдельную операцию

Основной долг закроется, затем откроется обычная операция на оставшуюся сумму.

Новая операция на остаток

Долг уже будет закрыт на его фактический остаток. Здесь записывается только отдельная сумма, которая к долгу не относится.
