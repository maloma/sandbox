# PF-08A-M3-06 — Mobile UX Reliability

Status: In review

Scope:
- reliable full-target short tap for obligation status;
- collapsed operation revision history by default;
- measured bottom reservation for fixed navigation and action dock;
- custom operation date picker with visibly distinct current and selected dates;
- regression proof for linking a previously created manual expense without creating a duplicate.

Out of scope:
- partial and multi-operation payments;
- planned incomes;
- push notifications.
